# Math Case — Кейс ДенВал по математике ЕГЭ

## Команды

### Разработка

| Команда | Действие |
|---|---|
| `pnpm run dev` | Запустить dev-сервер (Vite, http://localhost:5173/math_case/) |
| `pnpm run build` | Сборка для продакшена (запускает нормализацию через `prebuild`) |
| `pnpm run preview` | Превью продакшен-сборки локально |

### Задания

| Команда | Действие |
|---|---|
| `pnpm run tasks:sync` | Полный цикл: скачать с Я.Диска → нормализовать → обновить tasks.json → сгенерировать подсказки |
| `pnpm run tasks:pull` | Только скачать/обновить PNG из Яндекс.Диска в `public/tasks/` |
| `pnpm run normalize` | Только нормализовать все PNG в `public/tasks/` до размера 600×300 |
| `pnpm run tasks:gen-json` | Только пересобрать `tasks.json` по PNG-файлам в `public/tasks/` |
| `pnpm run tasks:gen-hints` | Только сгенерировать подсказки для заданий с пустым `message` (требует `OPENAI_API_KEY`) |

## Настройка окружения

Создай файл `.env` в корне проекта и заполни нужные переменные:

```bash
touch .env
```

### Синхронизация с Яндекс.Диском

**Вариант A — публичная ссылка (без токена):**
```
YADISK_PUBLIC_URL=https://disk.yandex.ru/d/<ключ>
YADISK_PUBLIC_PATH=/          # подпапка внутри ресурса, по умолчанию /
```

**Вариант B — OAuth-токен (приватная папка):**
```
YADISK_TOKEN=<токен>
YADISK_TASKS_DIR=/tasks       # путь к папке на Диске
```

Токен Яндекс.Диска можно получить по адресу: https://oauth.yandex.ru

### Автогенерация подсказок через OpenAI (опционально)

```
OPENAI_API_KEY=sk-...
```

Без ключа `tasks:gen-hints` выводит список заданий без подсказок и завершается без ошибки.

> Файл `.env` **не коммитить** — он в `.gitignore`.

## Структура скриптов

| Файл | Описание |
|---|---|
| `scripts/yadisk-pull-tasks.js` | Скачивание PNG из Яндекс.Диска (WebDAV или публичная ссылка) |
| `scripts/normalize-tasks.js` | Нормализация PNG до 600×300 (белый фон, контент по центру) |
| `scripts/gen-tasks-json.js` | Генерация `tasks.json` из PNG-файлов в `public/tasks/` |
| `scripts/gen-hints.js` | Генерация мотивирующих подсказок через GPT-4o Vision |

---

## Добавление новых заданий

1. Загрузи PNG-картинки в расшаренную папку на Яндекс.Диске
2. Запусти:
   ```bash
   pnpm run tasks:sync
   ```
3. Файлы скачаются, нормализуются до 600×300 и появятся в `tasks.json`
4. При необходимости отредактируй `message` в `public/tasks/tasks.json` вручную — при следующем `tasks:sync` оно сохранится
