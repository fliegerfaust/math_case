# Math Case Opener — Требования к проекту

## Внешний вид

- Нейтральный фон экрана (тёмный, нейтральный цвет)
- По центру экрана отображается кейс в хайтек-стиле (тёмный металл, неоновые глоуинг-края, пульсирующая анимация ожидания)

## Механика открытия

- По нажатию на кейс запускается анимация открытия
- Анимация — горизонтальный скроллинг полосы с заданиями ЕГЭ по математике (PNG-картинки)
- Стиль анимации — как на сайте https://case.oki.gg/case/operation-bravo-case/open:
  - Полоса из миниатюр заданий прокручивается справа налево
  - Начинается быстро, затем плавно замедляется (cubic-bezier easing)
  - В центре — вертикальный маркер с подсветкой
  - Итоговое задание останавливается у маркера

## Контент

- Задания представлены PNG-картинками
- Преподаватель может загружать свои картинки заданий (MVP: файлы кладутся в `public/tasks/`, описываются в `tasks.json`)
- Изначально используются картинки с сайта https://math-ege.sdamgia.ru/
- Каждое задание сопровождается мотивирующим текстом от преподавателя (хранится в `tasks.json`)

## После выпадения задания

- На экране показывается полный размер картинки задания с анимацией появления
- Под заданием отображается мотивирующий текст от преподавателя
- Есть кнопка «Открыть снова» — возврат к экрану с кейсом, при следующем открытии выпадает другое задание

## Развёртывание

- Приложение публикуется на **GitHub Pages**
- Доступ открытый — любой пользователь может открыть ссылку без авторизации
- Сборка: `vite build` с `base` настроенным под репозиторий GitHub Pages
- CI/CD: GitHub Actions workflow для автоматического деплоя при пуше в `main`

## Технический стек

- **React + Vite** (SPA)
- **Plain CSS** — без UI-фреймворков, полный контроль над анимациями
- Нет бэкенда — все данные статические (`public/tasks/`)
- `tasks.json` — конфиг заданий: `[{ id, image, message }]`
